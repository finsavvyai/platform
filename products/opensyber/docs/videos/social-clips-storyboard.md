## Production Notes

### Recording Checklist
- [ ] Browser: Arc or Chrome, dark mode, zoom 110%, hide bookmarks bar
- [ ] Resolution: 1080x1920 native (phone mirror or vertical monitor)
- [ ] Clean desktop: no notifications, no personal tabs visible
- [ ] Use demo account with realistic but non-sensitive data
- [ ] Record at native speed, speed up in editing (preserves quality)

### Font Stack (all videos)
| Role | Font | Weight | Usage |
|---|---|---|---|
| Display/Headlines | Bebas Neue | Regular | All-caps headlines, impact text |
| Code/Technical | Space Mono | Regular | Timestamps, URLs, terminal text |
| Body/Labels | DM Sans | Regular/Medium | Descriptions, captions, labels |

### Color Palette
| Token | Hex | Usage |
|---|---|---|
| Signal/Primary | `#00E5C3` | CTAs, highlights, checkmarks, URLs |
| Background | `#080B0F` | All backgrounds |
| Text Primary | `#FFFFFF` | Headlines |
| Text Secondary | `#A0A0A0` | Descriptions, captions |
| Alert/Red | `#FF4D4D` | Errors, attack indicators, X marks |
| Alert Soft | `#FF8080` | Left-side comparison items |
| Success Soft | `#80FFD0` | Right-side comparison items |

### Tools
- Screen recording: OBS Studio (vertical canvas 1080x1920)
- Editing: CapCut Pro or DaVinci Resolve
- Music: Suno AI (prompts above) or licensed trending audio
- Captions: CapCut auto-captions, then manually style to match font stack
- Thumbnail: Figma (export 1080x1920)

API Documentation

GET /api/datasources

List all available data sources.

POST /api/query/execute

Execute a natural language query against a data source.

{
    "text": "Show me the top 10 customers by revenue",
    "datasourceId": 1
}

GET /api/prompts

Get all common prompts. Supports query parameters:

  • category - Filter by category
  • search - Search in title, description, or prompt text

POST /api/prompts

Create a new common prompt.

{
    "title": "Count Records",
    "description": "Count all records in a table",
    "promptText": "Count all records",
    "category": "Basic Queries"
}

GET /api/prompts/favorites

Get all favorite prompts.

POST /api/prompts/{id}/favorite

Toggle favorite status of a prompt.

POST /api/prompts/{id}/use

Increment usage count of a prompt.

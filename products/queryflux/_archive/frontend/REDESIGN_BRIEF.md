# QueryFlux Marketing Website - Redesign Brief for Bolt.new

## 🎯 Project Overview
**QueryFlux** is an AI-powered database management platform that needs a professional marketing website to drive subscriptions and showcase product features.

## 🎨 Current State
- **Live URL**: https://queryflux-app.netlify.app
- **Current Issue**: Blank page, poor design, not loading properly
- **Tech Stack**: React 18 + TypeScript + Vite + Tailwind CSS
- **Project Structure**: Located in `/frontend` directory

## 🚀 Goal
Create a stunning, professional marketing website that:
- Converts visitors into paying customers
- Showcases AI-powered database management features
- Supports subscription sales (Starter $0, Pro $19/mo, Enterprise Custom)
- Drives downloads for desktop apps (Mac, Windows, VS Code)

## 📋 Required Pages & Sections

### 1. Landing Page (Homepage)
- **Hero Section**: 
  - Compelling headline "AI-Powered Database Manager"
  - Sub-headline about transforming database management
  - Download buttons (Mac, Windows, VS Code)
  - Feature highlights (multi-database, AI queries, collaboration)

- **Features Section**:
  - Multi-Database Support (35+ databases)
  - AI-Powered Queries (natural language to SQL)
  - Real-time Collaboration
  - Code Generation (APIs, ORMs)
  - Enterprise Security
  - Performance Monitoring

- **Pricing Section**:
  - Starter: $0/forever (3 databases, basic features)
  - Professional: $19/month (20 databases, AI features, team collaboration)
  - Enterprise: Custom (unlimited, advanced security, dedicated support)

- **Social Proof**:
  - Testimonials from users
  - Usage statistics (10,000+ users, 1M+ queries)
  - Trust badges and security certifications

- **CTA Section**:
  - Download buttons
  - Free trial signup for Professional plan

### 2. Navigation Header
- Logo and branding
- Navigation: Features, Pricing, Documentation, Account
- Download buttons
- Sign In/Account access
- Mobile responsive menu

### 3. Footer
- Company info and links
- Social media links
- Legal pages (Privacy, Terms, etc.)
- Newsletter signup

## 🎨 Design Requirements

### Visual Style
- **Modern & Professional**: Clean, enterprise-ready design
- **Color Scheme**: Purple/violet primary (#8B5CF6), with supporting colors
- **Typography**: Inter font family, clean hierarchy
- **Layout**: Clean sections, good use of whitespace, responsive grid

### Key Components
- **Header**: Sticky navigation with CTAs
- **Hero**: Full-width hero with gradient backgrounds
- **Feature Cards**: Hover effects, icons, descriptions
- **Pricing Cards**: Highlighted "Most Popular" plan
- **CTA Buttons**: Primary (purple), secondary (outlined), tertiary
- **Footer**: Multi-column layout with links and newsletter

### Interactive Elements
- Smooth hover states and transitions
- Mobile-responsive navigation
- Interactive pricing toggles (monthly/yearly)
- Newsletter signup form
- Download buttons with platform detection

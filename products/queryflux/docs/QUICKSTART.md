# QueryFlux QuickStart Guide

Welcome to QueryFlux! This guide will get you connected to your first database in under 5 minutes.

## Installation

### Desktop App (Recommended)
Download the latest version for your platform:

| Platform | Download |
|----------|----------|
| macOS | [QueryFlux-mac.dmg](https://github.com/queryflux/desktop/releases/latest) |
| Windows | [QueryFlux-win.exe](https://github.com/queryflux/desktop/releases/latest) |
| Linux | [QueryFlux-linux.AppImage](https://github.com/queryflux/desktop/releases/latest) |

### Web Version
Access QueryFlux directly in your browser at [app.queryflux.com](https://app.queryflux.com)

---

## Your First Connection

### Step 1: Click "New Connection"
From the Connections tab, click the **+ New Connection** button.

### Step 2: Choose Your Database Type
Select from 35+ supported databases:
- **SQL**: PostgreSQL, MySQL, SQLite, SQL Server, Oracle
- **NoSQL**: MongoDB, Redis, Cassandra
- **Cloud**: Supabase, PlanetScale, Neon, AWS RDS

### Step 3: Enter Connection Details
```
Host: localhost (or your server address)
Port: 5432 (default for PostgreSQL)
Database: your_database_name
Username: your_username
Password: ••••••••
SSL: Enable for production
```

### Step 4: Test & Save
Click **Test Connection** to verify, then **Save**.

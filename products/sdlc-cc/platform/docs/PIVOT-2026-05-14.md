# Pivot — 2026-05-14

## Decision

Abandon enterprise compliance positioning. After a same-day
competitor-research pass on the originally-proposed alternatives,
**only Path 6 survives**:

- **Path 4** — MCP server / Claude App — **KILLED.** Helicone already
  ships `@helicone/mcp` as a free→paid funnel; Nightfall / Lakera /
  Strac own enterprise DLP-MCP; <5% of MCP servers monetize.
- **Path 5** — Cost-ops microSaaS at $19/mo — **KILLED.**
  [OpenRouter Guardrails](https://openrouter.ai/docs/guides/features/guardrails)
  ships free per-key budget caps that reject requests; Helicone was
  acquired by Mintlify in March 2026.
- **Path 6** — AMLIQ component port — **CONFIRMED.** Internal port of
  `audit` / `RBAC` / `DLP` / `spend` into the **aegis** monorepo.
  Strengthens an already-monetising product. No external competitive
  surface.

## Why now

### Market re-audit (full assessment in [the archive](archive/2026-05-14-enterprise-pivot/competitive-analysis.md))

The "compliance LLM gateway + RAG + DLP" stack has 8+ real players
in May 2026. Hyperscalers (Cloudflare AI Gateway, AWS Bedrock
Guardrails, Azure AI Content Safety) bundle the core feature set
for ~free. Cisco bought Lakera in May 2025 and rolled it into
**Cisco AI Defense**, consolidating the security layer. Total
enterprise RAG stack costs $500-$2000/month all-in. The $5K/month
"Team" tier sits in the dead quadrant: above commodity, below
enterprise.

### Anthropic FSI launch (2026-05-05) — headwind, not tailwind

Anthropic announced 10 prebuilt finance agents, Claude Opus 4.7
for financial workloads, a **FIS partnership** for the AML / financial
crimes agent (BMO, Amalgamated Bank), and a **$1.5B JV with
Blackstone, Goldman Sachs**. Banks now buy AI direct from
Anthropic + FIS. A non-partner middleware gateway gets
disintermediated. The earlier `LEVERAGE_ANTHROPIC_FSI.md` strategy
inverted: this is competition, not co-sell.

### No regulation budget

The enterprise positioning required:

| Item | Cost |
|---|---|
| SOC 2 Type II audit | $30K-$60K |
| ISO 42001 certification | $30K-$100K |
| FINRA / EU AI Act content review (legal) | $20K-$50K |
| Enterprise sales motion (CAC) | $5K-$20K per win |

That capital is not available. Without it, the regulator-mapped
controls — which were the one remaining differentiator after the
market re-audit — cannot be productized and sold at the price needed
to fund them. Dead loop.

## What the three paths each fund

### Path 6 — AMLIQ port (now)

Port the four highest-leverage packages into the aegis monorepo where
the AMLIQ product can use them:

| Package | Aegis target |
|---|---|
| `services/gateway/internal/infrastructure/audit` | `internal/audit/` |
| `services/gateway/internal/domain/rbac` + `infrastructure/rbac` | `internal/auth/rbac/` |
| `services/gateway/internal/infrastructure/middleware/dlp_*` | `internal/security/dlp/` (alongside existing PAN/IBAN/BIC) |
| `services/gateway/internal/domain/spend` + `infrastructure/spend` | `internal/billing/spend/` |

Net effect: AMLIQ inherits the immutable audit log, RBAC evaluator
with `RequirePermission` middleware, the inbound + outbound DLP
chain, and the per-tenant spend tracker — without any new build
work. Effort: 1-2 weeks.

### Path 4 — MCP server — KILLED

Originally pitched as a free MCP server with three tools
(`redact_prompt`, `check_budget`, `record_call`) distributed via the
MCP registry, upselling to a hosted version. The 2026-05-14 research
pass invalidated every part of that pitch:

- **Helicone** already ships `@helicone/mcp` as a free server pointing
  back to their hosted product — same free→paid playbook, executed
  by a funded incumbent that was acquired by Mintlify in March 2026.
- **Nightfall, Lakera (Cisco), Strac** own the enterprise DLP-MCP
  slot. None has a Solo/SMB pricing tier, but their feature set
  defines what enterprise buyers expect.
- **`mcp-server-conceal`** (OSS) owns the free dev tier.
- The **top 20 MCP servers** by install count are all *tools*
  (GitHub: 28.3k stars, Figma Dev Mode, Notion, Sequential Thinking,
  wcgw). Zero policy / DLP / budget servers in the top 20.
- The ecosystem is large (97M SDK downloads / month in March 2026,
  20k+ servers on Glama) but **<5% of MCP servers monetize**. The
  one published free→paid case study (21st.dev, $10K MRR in 6 weeks)
  is a UI-component server, not a policy layer.

### Path 5 — Cost-ops microSaaS at $19/mo — KILLED

Originally pitched as multi-provider spend tracking + budget caps +
alert + monthly report at $19/mo, with "hard-stop 402 instead of
just observability dashboards" as the differentiator. The 2026-05-14
research pass invalidated the differentiator entirely:

- **[OpenRouter Guardrails](https://openrouter.ai/docs/guides/features/guardrails)**
  ships per-key and per-member daily / weekly / monthly budget caps
  that **reject requests when the limit is reached** — for free, on
  every API key. The "only one with hard stop" pitch is false.
- **Portkey** ships hard caps with block + throttle at the Pro /
  Enterprise tier.
- **Helicone** (observability-only, no hard cap) was acquired by
  Mintlify in March 2026 — the observability market is consolidating,
  not opening for a solo SKU.
- **Langfuse** has no request hard cap on paid tiers; bills overage
  instead.
- **Unit economics check** (5M tokens/mo user, Neon free tier,
  LemonSqueezy 5% + $0.50 fee): ~$17.55 gross margin per user.
  At 300 users: ~$5.3K MRR less infra. Only worth the solo effort if
  OpenRouter and Portkey don't already exist — which they do.

## What this repo is for now

`services/gateway` stops being a product. With Paths 4 and 5 killed,
it becomes a **sunset codebase** with three remaining roles:

- **Donor codebase** for Path 6 (AMLIQ port — the only active work)
- **Reference architecture** for the broader gateway design, kept so
  anyone re-opening the enterprise direction can read the archived
  decision tree
- **Cloudflare-Pages landing host** for `sdlc.cc` until the domain
  is repurposed or retired

The enterprise positioning, the `cmd/sdlc-api` aspirational binary,
the 90-day GA roadmap, the SOC 2 Type II target, and the original
"Paths 4 + 5 + 6" version of this very document — all archived under
[`docs/archive/2026-05-14-enterprise-pivot/`](archive/2026-05-14-enterprise-pivot/).

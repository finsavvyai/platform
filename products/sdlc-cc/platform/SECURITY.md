# Security Policy

## Reporting a Vulnerability
Report suspected vulnerabilities privately via repository security advisories or direct maintainer contact.
Do not open public issues for unpatched vulnerabilities.

## Scope
This policy covers all services, packages, and deployment assets in this repository.

## Response Targets
- Initial acknowledgement: within 24 hours
- Triage decision: within 72 hours
- Mitigation plan: within 7 days for high/critical issues

## Disclosure
Coordinated disclosure is expected. Public disclosure should happen only after a patch is available.

sdlc.cc — extension options

Sets the gateway your prompts get scrubbed against. Keys are scoped per-browser via chrome.storage.sync.

Gateway
Active on
  • claude.ai — prompt input + send shortcut
  • chatgpt.com / chat.openai.com — prompt input + send shortcut

Add more hosts in manifest.json + content.js HOSTS map. PRs welcome.

How it works
  • You press send (or Cmd/Ctrl-Enter).
  • Extension intercepts, sends the prompt to your gateway's /v1/dlp/scrub.
  • If anything was redacted, the input gets replaced with the scrubbed version and a notice — press send again to submit.
  • If nothing was redacted, the original prompt submits without your noticing.

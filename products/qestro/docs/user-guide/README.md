## 🔍 Quick Reference

### Keyboard Shortcuts
| Action | Windows/Linux | Mac |
|--------|---------------|-----|
| Start Recording | `Ctrl+Shift+R` | `Cmd+Shift+R` |
| Stop Recording | `Ctrl+Shift+S` | `Cmd+Shift+S` |
| Run Test | `Ctrl+Enter` | `Cmd+Enter` |
| Save Test | `Ctrl+S` | `Cmd+S` |
| Open Command Palette | `Ctrl+Shift+P` | `Cmd+Shift+P` |

### Common Commands
- `questro create project <name>` - Create new project
- `questro run test <test-id>` - Run specific test
- `questro record web <url>` - Start web recording
- `questro record mobile <app>` - Start mobile recording

### File Types
- `.questro` - Questro test file
- `.questro-config` - Project configuration
- `.questro-env` - Environment variables
- `.questro-data` - Test data files

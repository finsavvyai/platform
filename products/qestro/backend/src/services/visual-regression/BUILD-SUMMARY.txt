ARCHITECTURE DIAGRAMS
═════════════════════

Service Dependencies:
  
  Routes
    ↓
  VisualRegressionEngine
    ├── ScreenshotService → Playwright
    ├── ImageComparator → PNG pixels
    └── BaselineManager → Filesystem

Data Flow:
  
  User API Call
    ↓
  Route Handler (auth check)
    ↓
  VisualRegressionEngine.runVisualTest()
    ↓
  1. ScreenshotService.captureScreenshot()
  2. BaselineManager.getBaseline()
  3. ImageComparator.compareImages()
  4. Return VisualTestResult
    ↓
  Format Response (JSON)
    ↓
  Send to Client

Quick Installation

npm (Recommended)

npm install -g @questro/agent

Requires Node.js 16+ and npm

Direct Download

  1. Download the appropriate installer for your operating system
  2. Run the installer and follow the setup wizard
  3. Open terminal/command prompt and run questro-agent --version
  4. Authenticate with your Questro account using questro-agent login

Docker

docker run -it questro/agent:latest

For containerized environments

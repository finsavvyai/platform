## About Section

Qestro is an AI-powered testing automation platform built by FinsavvyAI. We help development teams ship faster with fewer bugs by automating the entire QA lifecycle -- from test creation to execution to maintenance.

Describe your test in plain English or paste a user story, and Qestro generates production-ready Playwright, Maestro, or API test code using large language models. No more writing boilerplate test scripts by hand.

What makes Qestro different:

- AI Test Generation: Turn user stories and requirements into executable test suites. Qestro analyzes your application and generates comprehensive browser, mobile, and API tests automatically.

- Self-Healing Assertions: When your UI changes, tests break. Qestro detects selector drift, timing issues, and assertion mismatches, then auto-repairs your tests so they keep passing without manual intervention.

- Cross-Platform Orchestration: One platform for browser testing (Playwright on Chromium, Firefox, WebKit), mobile testing (Maestro on iOS and Android), and API testing (REST and GraphQL with chaining, auth, and assertions).

- CI/CD Integration: Native GitHub Actions and GitLab CI support. Trigger test suites on every push, view results in your pipeline, and block merges on failure.

- Analytics and Flakiness Detection: Track pass rates, execution trends, and slowest tests. Qestro identifies flaky tests automatically and surfaces patterns so you can fix root causes.

- Enterprise Ready: Team workspaces, role-based access, audit logging, cron-based scheduling, parallel sharding, and report generation in JUnit XML, HTML, Allure, JSON, and CSV.

Built for QA engineers who want less maintenance, developers who want fast feedback loops, and engineering leaders who want visibility into quality metrics.

Pricing:
- Free: 5 projects, 100 test runs/month
- Starter: 50 projects, $99/month
- Pro: 500 projects, $499/month, includes mobile and API testing
- Enterprise: Unlimited projects, custom pricing, on-prem deployment option

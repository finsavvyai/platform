# Qestro -- Logo Generation Prompts

## Brand Identity

- **Primary Gradient**: Purple (#7c3aed) to Blue (#3b82f6)
- **Style**: Modern, clean, enterprise-grade, AI-powered
- **Key Element**: Emphasized "Q" letterform
- **Usage**: Must work at favicon size (16x16, 32x32) and large marketing sizes (1200px+)
- **Background**: Should work on both light and dark backgrounds

---

## Text Description

A modern, minimal logo for "Qestro" -- an AI-powered software testing platform. The logo features a stylized uppercase "Q" as the primary mark, rendered with a gradient flowing from deep purple (#7c3aed) on the left to bright blue (#3b82f6) on the right. The Q's tail is extended into a subtle checkmark or circuit-line motif, representing both quality assurance (the checkmark) and AI/technology (the circuit path). The letterform uses clean geometric construction with slightly rounded terminals for approachability. The full wordmark "estro" follows in a clean sans-serif typeface (similar to Inter or SF Pro) in neutral dark gray. At small sizes, only the Q mark is used as the icon.

## Variations to Generate

### 1. Icon Only (Favicon / App Icon)
- Just the stylized Q mark
- Must be legible at 16x16 pixels
- Use the full gradient
- Simple geometric form, no fine details

### 2. Horizontal Lockup (Header / Navigation)
- Q icon + "estro" wordmark to the right
- Wordmark in Inter Medium or SF Pro Medium, dark gray (#1f2937)
- Consistent vertical alignment

### 3. Stacked Lockup (Marketing / Social)
- Q icon centered above
- "QESTRO" wordmark below in spaced uppercase
- Used for social media avatars, presentation title slides

### 4. Monochrome (Print / Watermarks)
- Solid white version for dark backgrounds
- Solid dark (#1f2937) version for light backgrounds
- No gradient, single color

### 5. Animated (Web / Loading States)
- The Q tail draws itself as a checkmark animation
- Gradient sweeps from purple to blue during the draw
- Duration: 0.8s ease-out
- Use as loading indicator or success confirmation

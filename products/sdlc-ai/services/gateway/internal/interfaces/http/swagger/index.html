🚀 Quick Start Guide

  • 1
    Authenticate
    Login to get your JWT access token
    curl -X POST https://api.sdlc.ai/v1/auth/login \ -H "Content-Type: application/json" \ -d '{ "email": "user@example.com", "password": "your-password", "tenant_id": "your-tenant-id" }'
  • 2
    Upload Document
    Upload a document for processing
    curl -X POST https://api.sdlc.ai/v1/documents \ -H "Authorization: Bearer YOUR_JWT_TOKEN" \ -F "file=@document.pdf" \ -F 'metadata={"title": "My Document"}'
  • 3
    Query AI
    Ask questions about your documents
    curl -X POST https://api.sdlc.ai/v1/rag/query \ -H "Authorization: Bearer YOUR_JWT_TOKEN" \ -H "Content-Type: application/json" \ -d '{ "query": "What are the key points in this document?", "model": "gpt-4" }'

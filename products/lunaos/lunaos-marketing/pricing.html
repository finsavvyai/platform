Frequently asked questions

What counts as an "agent run"?
Each time you run luna run <agent> or trigger an agent via the dashboard or API, that counts as one run. Chaining multiple agents counts as one run per agent in the chain. Viewing or sharing a report does not count.
Do I need an account to use the CLI?
No. You can run npx luna-agents run code-review in any git project without an account. A free account unlocks run history, saved reports, and RAG indexing for your repo.
What LLMs do the agents use?
Agents use a multi-provider setup (OpenAI GPT-4o, Anthropic Claude, and Cloudflare Workers AI) based on the task. You can also bring your own API keys on Free tier to use your own models without burning LunaOS quota.
Can I cancel anytime?
Yes. Cancel anytime from your billing dashboard. You keep Pro access until the end of your paid period. No questions asked, no cancellation fee.
Is there an Enterprise plan?
Yes — Enterprise includes SSO, custom deployment, data residency (EU/US), SLA, and volume pricing. Contact us to discuss your requirements.
Where does my code go?
Code is processed transiently for analysis — we do not store your source code. RAG embeddings (vectors, not raw code) are stored in Cloudflare Vectorize. You can delete all data at any time via the settings page.

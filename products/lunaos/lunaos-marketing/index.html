FAQ

Questions, answered

Does my code leave my machine?

The CLI runs locally. RAG indexing sends embeddings (not raw source) to Cloudflare Vectorize in your region. Bring-your-own-key mode keeps inference calls direct to OpenAI/Anthropic — we never see your code or prompts.

Which LLM providers are supported?

OpenAI, Anthropic Claude, Workers AI, and local llamafile. Pick per-agent. BYOK on Free, managed keys on Pro/Team.

How is this different from GitHub Copilot or CodeRabbit?

Copilot autocompletes lines. CodeRabbit reviews PRs in GitHub. LunaOS runs 28 specialised agents — review, test-gen, security audit, deploy checks, docs — from one CLI, with RAG over your full repo. Works in any project, any CI, any IDE.

What languages and frameworks does it support?

TypeScript, JavaScript, Python, Go, Rust, Java, Ruby, PHP, C#, Swift, Kotlin. Test generation supports Jest, Vitest, pytest, Go test, JUnit. New frameworks added monthly.

Can I self-host?

The CLI is open source — self-host the agent runner today. Hosted Studio + Dashboard self-host is on the roadmap for Q3 2026 (Team plan).

What's the refund policy?

30-day money-back on Pro and Team. Cancel anytime from the dashboard. No usage lock-in — export your data via API.

Is there a CI/CD integration?

Yes. Two lines in GitHub Actions, GitLab CI, or any runner. Block merges on critical findings, post review comments to PRs, write artifacts to S3/R2.

Free tier limits?

1 project, 500 agent runs / month, community Discord. Pro lifts both caps. No surprise overage charges — you hit the cap, runs queue until next cycle or you upgrade.

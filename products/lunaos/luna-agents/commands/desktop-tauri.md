# /desktop-tauri — HIG Desktop Scaffold

Shortcut for `/ll-desktop-tauri`. Generates Tauri 2.0 + shadcn/ui app with native menus, code signing, and HIG/Fluent/GNOME/Minimal design.

```
/desktop-tauri MyApp ai.example.myapp                  # default HIG
/desktop-tauri MyApp ai.example.myapp --design fluent  # Windows
/desktop-tauri MyApp ai.example.myapp --target macos   # mac only
```

# /skill-radar — New-Skill Scanner

Shortcut for `/ll-skill-radar`. Scans curated lists weekly, scores by stars + recency + relevance, surfaces what's new.

```
/skill-radar                              # all categories, top 20
/skill-radar performance --min-stars 100  # perf only
/skill-radar all --install 3              # auto-PR top 3
```

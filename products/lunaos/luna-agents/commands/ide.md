# /ide — IDE Plugin Generator

Shortcut for `/ll-ide-plugins`. Scaffold every IDE/editor plugin at once with shared core.

Examples:
```
/ide all
/ide cursor,windsurf,vscode
/ide vscode-forks,jetbrains-all
```

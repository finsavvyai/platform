LemonSqueezy Product Setup

Click any field to copy its value to your clipboard. Then paste into LemonSqueezy.

Setup Instructions
  1. Go to LemonSqueezy Dashboard > Products > New Product.
  2. Copy the Product Name and Product Description from the first section below.
  3. Create the product, then add two variants (Pro and Team) with the fields below.
  4. Set billing to Subscription / Monthly for both variants.
  5. Use the AI Image Prompt with your preferred image generator to create product card images (800x400px).
  6. The Free tier is documented here for reference only -- it does not need a LemonSqueezy product.

Webhook Configuration

Required after product creation
LemonSqueezy Webhook
Go to Settings → Webhooks → Add Webhook
Webhook URL
https://api.lunaos.ai/billing/webhook
Signing Secret
Generate in LemonSqueezy, then set via: wrangler secret put LEMONSQUEEZY_WEBHOOK_SECRET --name lunaos-engine
Events to Subscribe (select all of these)
  • subscription_created
  • subscription_updated
  • subscription_cancelled
  • subscription_expired
  • subscription_payment_failed
  • subscription_payment_success
Cloudflare Secrets
Run these after creating products and webhook
Set API Key
wrangler secret put LEMONSQUEEZY_API_KEY --name lunaos-engine
Set Store ID
wrangler secret put LEMONSQUEEZY_STORE_ID --name lunaos-engine
Set Webhook Secret
wrangler secret put LEMONSQUEEZY_WEBHOOK_SECRET --name lunaos-engine
List Variant IDs (run this to find your IDs)
curl -s https://api.lemonsqueezy.com/v1/variants -H "Authorization: Bearer $LEMONSQUEEZY_API_KEY" -H "Accept: application/vnd.api+json" | jq '.data[] | {id: .id, name: .attributes.name, price: .attributes.price}'
Set Pro Variant ID (use the id from above)
wrangler secret put LEMONSQUEEZY_VARIANT_PRO --name lunaos-engine
Set Team Variant ID (use the id from above)
wrangler secret put LEMONSQUEEZY_VARIANT_TEAM --name lunaos-engine
Run DB Migration
wrangler d1 execute lunaos-engine-db --remote --file=migrations/007_rename_stripe_to_ls.sql
Verify Health
curl -s https://api.lunaos.ai/health | jq .

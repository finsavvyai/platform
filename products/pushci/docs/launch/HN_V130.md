# Show HN: PushCI v1.3.0 — AI CI/CD that supports 33 languages and runs on your machine for $0

PushCI is an open-source CI/CD tool that runs on your own machine instead of rented cloud minutes. You run `pushci init`, it detects your stack, and `git push` triggers tests locally via a pre-push hook.

I built it because GitHub Actions charges $0.008/min and only meaningfully supports a handful of languages. Our team had Go, Rust, Terraform, and Kotlin — half of it went through CI unvalidated.

What's new in v1.3.0:

- **33 language stacks** — Go, Rust, Java, C#, Ruby, PHP, Swift, Dart, Elixir, Zig, Kotlin, Terraform, Helm, Solidity, Fortran, and 18 more. All with tested pipelines.
- **Perfetto performance tracing** — `pushci run --trace` generates Chrome Trace Event JSON. Load it in ui.perfetto.dev to find exactly where time is spent.
- **Flaky test detection** — `pushci run --stress 10` runs each check N times and reports flake rates.
- **Parallel agents** — build, test, scan, deploy on isolated git worktrees simultaneously. Race or consensus strategies.
- **5 notification channels** — Slack, Discord, Email, Telegram, Webhook in YAML config.
- **Llamafile lifecycle** — local LLM for AI pipeline diagnosis. No code leaves your machine.
- **Tailscale fleet discovery** — distributed runners across your own infrastructure via MagicDNS.

Tech: Go CLI (22 commands), React dashboard, Cloudflare Workers API (Hono + D1 + KV). 69 marketplace skills, 57 integration tests, 11 E2E tests.

Free tier: unlimited repos, unlimited local runs, all 33 stacks, tracing, flaky detection. $0 forever.

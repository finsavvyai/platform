## Tagline (60 chars max)
AI-native CI/CD. Zero config. Zero cost. Free forever.

## Description
PushCI replaces GitHub Actions with a single command: `npx pushci init`

AI auto-detects your tech stack (19 languages, 40+ frameworks), generates a complete CI/CD pipeline, and runs everything on your local machine for free.

No YAML. No cloud bills. No vendor lock-in.

**Key features:**
- 30-second setup (vs 30+ minutes for GitHub Actions)
- $0/month (vs $0.008/min for cloud CI)
- Works with GitHub, GitLab, AND Bitbucket
- Self-healing pipelines with AI root cause analysis
- MCP server for Claude Code, Cursor, Windsurf
- 16 built-in deploy targets (AWS, GCP, Azure, Vercel, etc.)

## First Comment
Hey Product Hunt! I built PushCI because I was tired of writing 50-line YAML files to run `npm test`.

The idea: what if CI/CD just worked? No config, no cloud account, no per-minute billing.

So I built an AI that scans your repo, detects your languages/frameworks/tests, and generates a complete pipeline. It runs on YOUR machine, so it's free forever.

Try it: `npx pushci init`

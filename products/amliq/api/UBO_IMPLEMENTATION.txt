3. HTTP API HANDLERS
===================

File: api/handler_ubo.go (Updated)
- UBOHandler struct maintains storage.UBORepository dependency
- AddOwner(w, r) - POST /api/v1/ubo - creates beneficial owner (201 on success)
- ListByOrg(w, r) - GET /api/v1/ubo/{id} - lists owners by organization with total ownership %
- DeleteOwner(w, r) - DELETE /api/v1/ubo/{owner_id} - removes owner (NEW)
  * Returns 400 if owner_id missing
  * Returns 200 with deleted ID on success
  * Returns 500 on database error

File: api/handler_ubo_screen.go (New)
- UBOScreenHandler with UBORepository and ScreeningRepository dependencies
- ScreenChain(w, r) - POST /api/v1/ubo/{id}/screen - screens ownership chain
  * Fetches all owners for organization
  * Returns array of UBOScreenResult per owner
  * Result fields: OwnerID, OwnerName, OwnershipPct, Matches, HighestConfidence, Flagged

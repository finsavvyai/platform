## Competitor Comparison

### vs Microsoft Lighthouse

Microsoft Lighthouse is a free multi-tenant management portal bundled with Microsoft 365. It provides visibility but lacks operational depth.

| Capability | Microsoft Lighthouse | TenantIQ |
|---|---|---|
| Multi-tenant view | Basic tenant list, limited filtering | Full control plane with cross-tenant search, filtering, and comparison |
| Security insights | Surface-level Secure Score summary | Deep CIS benchmark analysis with 100+ controls and trend tracking |
| Remediation | Manual -- links to individual admin portals | One-click remediation with rollback and audit trail |
| Compliance frameworks | None built-in | CIS, NIST, SOC 2, HIPAA mapping with automated evidence collection |
| License optimization | Basic license count visibility | AI-powered waste detection, reallocation recommendations, savings tracking |
| AI agent | None | Natural language queries, automated investigation, remediation suggestions |
| Alerting | Basic alerts, no customization | Real-time anomaly detection with severity scoring and escalation workflows |
| Reporting | Minimal | Executive-ready PDF/CSV exports with trend analysis |

**TenantIQ wins on**: Operational vs. informational. Lighthouse shows you the problem; TenantIQ fixes it.

## Competitive Moats

1. **Multi-tenant MSP-native architecture** -- Built from day one for MSPs managing dozens of tenants, not retrofitted from single-tenant SaaS.

2. **AI that executes, not just summarizes** -- Claude-powered agent that investigates, recommends, and remediates -- not a chatbot wrapper on existing dashboards.

3. **Remediation with rollback** -- One-click fixes with preview, approval workflows, and automatic rollback. Competitors offer read-only dashboards or manual scripts.

4. **Compliance automation, not checklists** -- Automated CIS benchmark scanning, drift detection, and evidence collection. Competitors provide static checklists that require manual verification.

5. **Edge-native performance** -- Deployed on Cloudflare Workers for sub-100ms response times globally. No cold starts, no regional latency, no infrastructure management.

## 🚀 USAGE EXAMPLES

### Example 1: Research Assistant

```bash
# Upload research papers
curl -X POST http://localhost:8080/api/notebook/sources/upload \
  -F "file=@paper1.pdf"
curl -X POST http://localhost:8080/api/notebook/sources/upload \
  -F "file=@paper2.pdf"

# Create research notebook
curl -X POST http://localhost:8080/api/notebook/notebooks \
  -H "Content-Type: application/json" \
  -d '{"name": "Literature Review"}'

# Query across papers
curl -X POST http://localhost:8080/api/notebook/query \
  -H "Content-Type: application/json" \
  -d '{
    "query": "What are the key differences between the approaches?",
    "source_ids": ["source_abc", "source_def"]
  }'
```

### Example 2: Code Documentation

```bash
# Upload code files
curl -X POST http://localhost:8080/api/notebook/sources/upload \
  -F "file=@main.py"
curl -X POST http://localhost:8080/api/notebook/sources/upload \
  -F "file=@README.md"

# Create documentation notebook
curl -X POST http://localhost:8080/api/notebook/notebooks \
  -H "Content-Type: application/json" \
  -d '{"name": "API Documentation"}'

# Ask questions about your code
curl -X POST http://localhost:8080/api/notebook/query \
  -H "Content-Type: application/json" \
  -d '{
    "query": "How does the authentication work?",
    "source_ids": ["source_abc"]
  }'
```

### Example 3: Multi-Node Cluster

```bash
# Machine A: LM Studio + Llama 3
python -m src.workers.worker_node --provider lmstudio

# Machine B: LM Studio + Mistral
python -m src.workers.worker_node --provider lmstudio

# Start cluster manager
python -m src.cluster.manager

# Create notebook with RAG across all nodes
curl -X POST http://localhost:8080/api/notebook/notebooks \
  -H "Content-Type: application/json" \
  -d '{"name": "Distributed Research"}'

# Query with load balancing
curl -X POST http://localhost:8080/api/notebook/query \
  -H "Content-Type: application/json" \
  -d '{
    "query": "Compare the approaches",
    "source_ids": ["source_1", "source_2", "source_3"]
  }'
```

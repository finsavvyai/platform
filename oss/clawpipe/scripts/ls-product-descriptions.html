Open app.lemonsqueezy.com/products/new in another tab. Click any field below to copy.

How to use

  1. Click any value to copy it to clipboard.
  2. In LemonSqueezy, click Create product, paste Name + Description.
  3. Add 3 variants (Dev, Growth, Scale). For each: copy Variant Name, Price, Description, paste Feature list as bullets.
  4. Generate hero / icon / OG / thumbnail images in your AI image tool — copy and paste each prompt.
  5. Once published, copy each variant's numeric ID from the dashboard URL into .env as LEMONSQUEEZY_VARIANT_DEV / GROWTH / SCALE.

ClawPipe is the intelligent AI pipeline that sits between your app and LLM providers. The only gateway that skips LLM calls entirely on deterministic prompts (Booster), then runs everything else through a self-learning router across 21 providers — with semantic caching and context packing on the way out. Cut your LLM bill 30–50% without changing a line of application logic.

# llamafile pins for the pipewarden-airgap goreleaser variant.
#
# Format: <arch>\t<filename>\t<sha256>\t<url>
# Lines starting with `#` and blank lines are skipped.
#
# Populate by running:
#   bash scripts/refresh-llamafile-checksums.sh 1
# and translating the resulting Go map literal into rows here. We keep
# this file empty in-tree to avoid baking a 4 GB artifact reference into
# the repo before the operator has reviewed which release to pin.
#
# Once populated, `make airgap-stage` materializes dist/airgap/llamafile-{arch}
# and `goreleaser release` packages them via the `airgap` archive id.

Integration:
============

1. vault initialized in main.go from PIPEWARDEN_VAULT_KEY env var
2. vault passed to router during initialization
3. router passes vault to handlers
4. handlers use vault.Encrypt() before DB storage
5. handlers use vault.Decrypt() after DB retrieval
6. LoadConnectionsFromDB() decrypts credentials at startup

All integration points use dependency injection pattern for testability.

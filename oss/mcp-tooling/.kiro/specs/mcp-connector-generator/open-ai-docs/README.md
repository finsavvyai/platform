# 🌍 MCPoverflow
A developer platform that auto-generates, hosts, and syncs MCP connectors from any API definition (OpenAPI / GraphQL / Postman).
---
## Quickstart
1. Clone the repo
2. Build with Go and TinyGo
3. Deploy via Cloudflare Workers + Pages
4. Access your dashboard at https://app.mcpoverflow.io

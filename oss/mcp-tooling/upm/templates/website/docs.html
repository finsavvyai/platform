Installation

Get UPM up and running in minutes.

Maven

<dependency>
    <groupId>com.upm</groupId>
    <artifactId>upm-core</artifactId>
    <version>1.0.0</version>
</dependency>

Gradle

implementation 'com.upm:upm-core:1.0.0'

npm

npm install @upm/core

pip

pip install upm

Getting Started

Create your first cross-language dependency in 5 minutes.

1. Create upm.yml

dependencies:
  python:
    - "requests:2.28.1"
    - "pandas:1.5.3"
  javascript:
    - "axios:1.6.0"

2. Use in Your Code

// Java
UPM upm = new UPM();
String result = upm.executePython(
    "import requests\n" +
    "r = requests.get('https://api.example.com')\n" +
    "return r.json()"
);

Examples

Java Using Python

import com.upm.UPM;

public class Example {
    public static void main(String[] args) {
        UPM upm = new UPM();
        
        // Use Python's requests library
        String response = upm.executePython(
            "import requests\n" +
            "r = requests.get('https://api.example.com/data')\n" +
            "return r.text"
        );
        
        System.out.println(response);
    }
}

Python Using JavaScript

from upm import UPM

upm = UPM()

# Use JavaScript's axios
result = upm.execute_javascript("""
    const axios = require('axios');
    const response = await axios.get('https://api.example.com');
    return response.data;
""")

print(result)

Configuration

Customize UPM for your needs.

upm.yml

upm_platform:
  base_url: "https://upmplus.dev"
  api_key: "your-api-key"
  timeout: 30000
  retry_attempts: 3

dependencies:
  python:
    - "requests:2.28.1"
  javascript:
    - "axios:1.6.0"

## Press Releases

### Release 1: UPM Emerges from Stealth with $5M Series A

**SAN FRANCISCO** – February 14, 2024 – UPM today emerged from stealth with $5 million in Series A funding to revolutionize software supply chain security. The round was led by [Venture Capital] with participation from [Angel Investors].

UPM's Universal Dependency Platform provides the first unified solution for securing software dependencies across all programming ecosystems. Unlike fragmented tools that focus on individual languages, UPM offers universal scanning, AI-powered risk prioritization, and one-click automated remediation.

"The software supply chain is the new frontier of cybersecurity," said Sarah Chen, CEO of UPM. "With 97% of applications containing open-source code and over 10,000 new CVEs published annually, organizations need a comprehensive solution. UPM doesn't just find vulnerabilities—we fix them automatically."

Key features include:
- Multi-ecosystem support (Maven, npm, PyPI, Cargo, Go, and more)
- AI-powered vulnerability prioritization
- Automated remediation with breaking-change detection
- Real-time IDE integration (IntelliJ, VS Code)
- Enterprise features (SSO, RBAC, audit logging)

UPM is currently used by 5,000+ organizations globally, including Fortune 500 companies in financial services, healthcare, and technology.

### Release 2: UPM Achieves SOC 2 Type II Certification

**SAN FRANCISCO** – February 14, 2024 – UPM today announced it has achieved SOC 2 Type II certification, demonstrating its commitment to security and compliance for enterprise customers.

The certification validates UPM's rigorous security controls across data protection, access management, and operational procedures. Enterprise customers can now deploy UPM with confidence knowing their dependency data is handled according to industry best practices.

### Release 3: UPM Launches Open Source Program

**SAN FRANCISCO** – February 14, 2024 – UPM today announced its open-source program, providing free access to the UPM platform for qualifying open-source projects.

The program includes:
- Free unlimited scanning
- Public vulnerability badges
- Community support
- Featured placement in UPM's directory

"Open source powers the world, and we want to give back," said Marcus Williams, CTO of UPM. "Securing the open-source ecosystem is part of our mission."

## Frequently Asked Questions

### Q: What problem does UPM solve?

A: UPM solves the software supply chain security crisis. Modern applications depend on thousands of open-source packages, many containing vulnerabilities. UPM provides the first unified platform to scan, analyze, and automatically remediate these vulnerabilities across all programming ecosystems.

### Q: How is UPM different from competitors?

A: Unlike single-ecosystem tools like Snyk or Dependabot, UPM supports all programming languages in one platform. Our AI-powered risk scoring reduces false positives by 87%, and our automated remediation can fix vulnerabilities in minutes instead of weeks.

### Q: Who are your customers?

A: We serve organizations of all sizes, from startups to Fortune 500 companies. Our customers include financial institutions, healthcare providers, e-commerce platforms, and technology companies—anyone building software with dependencies.

### Q: How much does UPM cost?

A: UPM offers a free tier for individuals and open-source projects. Team plans start at $49 per user per month. Enterprise pricing is custom based on organization size and requirements.

### Q: Is UPM self-hosted or SaaS?

A: Both! UPM offers a cloud-hosted SaaS solution for quick setup and a self-hosted enterprise option for organizations with strict data residency requirements.

## License Keys

Your license key unlocks features based on your tier:

```bash
# Activate your license
upm license activate UPM-XXXX-XXXX-XXXX-XXXX

# Check what features you have
upm license features

# View license details
upm license status
```

### License Key Format

UPM license keys follow this format:
```
UPM-TIER-ORGANIZATION-ID-CHECKSUM
```

Example:
```
UPM-PRO-ACME-CORP-12345-A7F3E9D2
```

# Vendored Open Source

This directory contains third-party open-source projects kept for reference,
integration research, or local development support.

These projects are not FinsavvyAI products and should not be represented as
owned IP. Keep upstream license files with each vendored project. When a
vendored project is no longer needed, remove it through an explicit cleanup
task rather than folding it into product code.

## Current execution status

The May 2026 consolidation plan moved the old open-source holding area here
so the root product inventory stays focused on FinsavvyAI-owned assets.
